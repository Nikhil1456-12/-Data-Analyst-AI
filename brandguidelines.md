# 🎨 AI Data Analyst App – Brand Guidelines (brandguidelines.md)

## 🎯 Product Vision
Build a clean, modern AI-powered data analyst tool that feels like a combination of ChatGPT + Power BI.

---

# 🧱 1. Core Experience

## User Journey
1. User enters query
2. AI processes request
3. Results displayed as:
   - SQL query
   - Table
   - Insights
   - Chart

---

# 💬 2. Chat Interface

## Layout
- Left panel → Chat interface
- Right panel → Results display

## Features
- Smooth conversation flow
- Message history
- Typing indicator

---

# 📊 3. Results Display

## Components
- SQL Query Section
- Data Table
- Insights Panel
- Chart Visualization

## Behavior
- Dynamic updates
- Scrollable results
- Export options (future)

---

# 🎨 4. UI Design Principles

## Style
- Clean and minimal
- Professional dashboard feel

## Colors
- Primary: Blue / Indigo
- Secondary: White / Light Gray
- Accent: Green (success), Red (error)

## Typography
- Headings: Bold
- Body: Clean readable font

---

# 📱 5. Responsiveness

- Mobile-friendly layout
- Adaptive grid system
- Smooth transitions

---

# 📈 6. Visualization Guidelines

## Chart Types
- Bar charts → category comparison
- Line charts → trends over time

## Design Rules
- Clear labels
- Tooltips
- Minimal clutter

---

# 🧠 7. AI Interaction Design

## Tone
- Professional
- Simple
- Insightful

## Response Format
1. SQL Query
2. Table Output
3. Insights (3–5 points)
4. Chart (if applicable)

---

# ⚙️ 8. Feature Expansion (Future)

- Dashboard builder
- Saved reports
- Power BI-like analytics
- AI recommendations

---

# 🔐 9. Trust & Safety

- Show SQL query transparency
- Prevent destructive operations
- Clear error messages

---

# 🚀 10. Performance

- Fast response time (<3 seconds)
- Smooth UI transitions
- Efficient rendering

---

# 🧩 11. Branding Feel

The application should feel like:
- Smart assistant
- Business intelligence tool
- Data expert

---

# ✅ Summary

This product should combine:
- Chat simplicity
- Dashboard power
- AI intelligence

To deliver a seamless data analytics experience.
